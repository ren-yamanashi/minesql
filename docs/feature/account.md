# アカウント

| 機能 | 実装 | 備考 |
| ---- | ---- | ---- |
| アカウント管理 | ✅ | 初期アカウントのみ。パスワード設定なども可能 |
| 認証プラグイン | ✅ | caching_sha2_password のみ (Fast Auth + Complete Auth) |
| ホスト名検証 | ✅ | 完全一致、サブネットパターン (`192.168.1.%`)、`%` (全許可) に対応 |
| TLS | ✅ | 全接続で TLS 必須。自己署名証明書を自動生成 |
| ソルト付きハッシュ | ✅ | MySQL 互換の `$A$005$` 形式 (SHA-256, 5000 回イテレーション) |
| アカウント作成 | - | - |
| パスワード変更 | ✅ | `ALTER USER 'user'@'host' IDENTIFIED BY 'new_password'` |
| アカウント削除 | - | - |
