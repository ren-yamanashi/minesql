# CREATE TABLE

| 機能 | 実装 | 備考 |
| ---- | --- | ---- |
| スキーマ名の指定 | - | - |
| テーブル名の指定 | ✅ | 実態は `${table_name}.db` になる |
| カラムのデータ型指定 | - | 全部文字列型として扱う |
| デフォルト値の指定 | - | - |
| NOT NULL 制約 | - | - |
| 外部キー制約 | - | - |

### テーブル (クラスタ化インデックス)

| 機能 | 実装 | 備考 |
| ---- | --- | ---- |
| テーブル名の指定 | ✅ | 実態は `${table_name}.db` になる |
| 複合プライマリキーの指定 | ✅ | Memcomparable format を使用してエンコード |
