# メタページ

## 概要

- B+Tree のルートノードの PageId を管理するためのページ
- B+Tree は成長するとルートが変化する可能性があるが、メタページに常に最新のルートノードの PageId を記録しておくことで、常にルートノードにアクセスできる

## ヘッダーの構成

- サイズ: 8 バイト
  - ルートノードの PageId を格納する
  - PageId は FileId (4 バイト) + PageNumber (4 バイト) で構成される

メタページの先頭 8 バイトを読み取ることで、ルートノードの PageId を取得できる
ルートノードが変化した場合は、メタページの先頭 8 バイトを書き換える
