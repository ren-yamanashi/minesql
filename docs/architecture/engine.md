# エンジン

## 概要

- サーバー起動時にストレージ層のリソースを初期化し、グローバルに保持する
- 保持するリソースは以下の 2 つ
  - [バッファプール](./storage/buffer/bufferpool.md)
  - [カタログ](./catalog.md)
- エグゼキュータやプランナーからグローバルに参照される

## 初期化の流れ

1. データディレクトリ (`MINESQL_DATA_DIR`) を作成する (存在しない場合)
2. バッファプールを生成する (`MINESQL_BUFFER_SIZE` で容量を指定)
3. カタログを初期化する
   - カタログファイル (`minesql.db`) が存在する場合: 既存のカタログを開き、登録済みテーブルの Disk をバッファプールに登録する
   - カタログファイルが存在しない場合: 新しいカタログを作成する
   - カタログファイルが空の場合: ファイルを削除して再作成する
